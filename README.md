
# 🎓 Học Bảng Nhân Chia (v4.0) - Trí Tuệ Nhân Tạo Gemini

Ứng dụng học toán thông minh dành cho học sinh tiểu học, tích hợp công nghệ **Gemini AI** để tạo môi trường học tập sinh động và cá nhân hóa.

## 🌟 Tính năng nổi bật

### 1. 🎙️ Hệ Thống Giọng Đọc Độc Quyền
- **Gemini API Voice (TTS):** Sử dụng model `gemini-2.5-flash-preview-tts` để tạo ra giọng đọc tự nhiên, chuẩn xác cho từng phép tính. **Lưu ý:** Ứng dụng không sử dụng giọng đọc máy mặc định của trình duyệt để đảm bảo chất lượng âm thanh tốt nhất.
- **Tải lên MP3 cá nhân:** Phụ huynh có thể tải tệp âm thanh (.mp3) ghi âm giọng đọc của chính mình hoặc của thầy cô. Em có thể học bảng cửu chương bằng những giọng nói thân thuộc nhất.

### 2. ✨ Minh Họa Trực Quan
- **Giải Thích Thông Minh:** AI hỗ trợ giải thích các phép tính thông qua sơ đồ nhóm ngôi sao sinh động, giúp học sinh hiểu rõ bản chất phép nhân và phép chia thay vì chỉ học thuộc lòng.

### 3. 📊 Luyện Tập Đa Dạng & Thành Tích
- Hệ thống bài tập phong phú: Tính toán, Điền số, So sánh, Đúng/Sai, Chọn dấu, Toán đố có lời văn.
- Phân loại độ khó và theo dõi tiến trình học tập qua bảng Thành Tích với các danh hiệu hấp dẫn như "Kiện Tướng", "Thần Đồng".

## 🛠 Hướng dẫn thiết lập âm thanh

1. **Nghe bằng giọng AI:** 
   - Vào **Cài Đặt** ⚙️ -> Tab **Chung** -> Chọn **Giọng AI**. 
   - Lựa chọn nhân vật **Cô giáo** hoặc **Thầy giáo** tùy ý.
2. **Nghe bằng giọng tự ghi (MP3):**
   - Vào **Cài Đặt** ⚙️ -> Tab **Giọng tự ghi**.
   - Nhấn biểu tượng 📤 để tải lên tệp MP3 cho từng bảng tính tương ứng.
   - Chuyển chế độ sang **Giọng Tự Ghi** để ứng dụng ưu tiên phát file MP3 của bạn.
